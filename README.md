# AI-Hotel-Concierge
AI Hotel Concierge that supports customers with common queries.

## Features
- Answering questions about amenities
- Answering questions about restaurant hours
- Answering questions about check in times
   
## Architecture and Technologies
  - Azure AI Foundry
  - AI Model: gpt-4.1-mini
  - Instructions: You are the virtual concierge for a hotel. Your tone should be warm, professional, and helpful. Always greet warmly and end conversations by asking if there's anything else you can help with.
